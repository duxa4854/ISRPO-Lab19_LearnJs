## Описание (что изучили)

В ходе выполнения лабораторной работы были изучены базовые концепции языка программирования JavaScript и проведено сравнение с языком C#. Основные темы, рассмотренные в работе:

- **Подготовка окружения:** Настройка рабочего пространства в VS Code и инициализация Git-репозитория.
- **Циклы:** Работа с циклами `for`, `while`, `do...while`, а также операторами `break` и `continue`. Синтаксис практически идентичен C#, за исключением отсутствия явного указания типов переменных.
- **Функции:** Изучены различные способы объявления функций (классические, стрелочные функции), параметры по умолчанию, замыкания (closures) и передача функций как значений (аналог делегатов).
- **Массивы:** Создание массивов, доступ к элементам, методы добавления/удаления (`push`, `pop`) и перебор элементов через `for...of`. Особенность JS — возможность хранения разных типов данных в одном массиве.
- **Объекты:** Работа с объектами (Object Literals), динамическое добавление свойств и методов, перебор свойств через `for...in`.

## Структура проекта

Проект состоит из следующих файлов:

- `index.html` — HTML-каркас страницы.
- `main.js` — Основной файл с JavaScript кодом (циклы, функции, массивы, объекты).
- `img/` — Папка для скриншотов выполнения заданий.
- `.git/` — Директория Git-репозитория.
